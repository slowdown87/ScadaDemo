## 2. 重大工艺变更 (v2.0 - 2026-04-28)

### 2.1 过滤工艺变更

**变更原因**: 膜过滤相比硅藻土过滤具有以下优势：
- 无硅藻土消耗，降低运行成本
- 无废硅藻土处理问题
- 过滤精度更高，产品澄清度更好
- 可实现CIP在线清洗

| # | 变更项 | 原方案 | 新方案 | 影响说明 |
|---|--------|--------|--------|----------|
| 9 | **过滤工艺** | 硅藻土过滤 + 板框压滤 | **膜过滤（超滤/微滤）** | 去除硅藻土过滤系统，简化过滤工艺 |

**原工艺流程**:
```
茶汁(80℃) → 硅藻土过滤 → 冷却沉淀 → 板框压滤 → 超滤膜 → 茶浓缩汁
```

**新工艺流程**:
```
茶汁(80℃) → 碟式离心机(预过滤) → 超滤膜(0.1μm) → 茶浓缩汁
```

**取消的设备**:
- FL-101 硅藻土过滤机
- FL-102 板框压滤机

**保留/新增的设备**:
- FL-101 碟式离心机（替换原硅藻土过滤机）
- FL-102 超滤膜组件
- FL-103 超滤循环泵

### 2.2 变更影响评估

| 系统 | 影响内容 |
|------|----------|
| 设备清单 | 删除硅藻土过滤系统设备，增加膜过滤设备 |
| 位号编码 | FL工段重新定义（见位号变更记录） |
| 控制逻辑 | 简化过滤工艺联锁 |
| SCADA点表 | 删除相关点位，新增膜过滤监控点位 |

## 3. 位号编码变更记录

### 3.1 电机编号变更

| 原编号 | 新编号 | 说明 |
|--------|--------|------|
| TH-102-MT | M-TH01 | 热交换机电机统一M-前缀 |

### 3.2 阀门编号统一

- 统一为 `WV/EV/BV/UV/PV/CV` 前缀
- 避免歧义，便于SCADA组态

### 3.3 分析仪表后缀统一

- `pHT/BT/CtT` 后缀格式统一
- 与国际惯例一致

## 4. 监控点位变更记录

### 4.1 新增点位

| 增加类型 | 新增位号 | 说明 |
|----------|----------|------|
| 超滤反洗监控 | FL-102-DPT, FL-102-FT2 | 膜压差监测、浓水流量 |
| 糖浆溶解监控 | BL-102-TT | 糖浆温度 |
| 灯检工段监控 | PK-102-ST, PK-102-DT | 速度和状态 |
| 计数点位 | CNT-007~CNT-011 | 细化的产量统计 |

### 4.2 点位变更影响

| 系统 | 影响内容 |
|------|----------|
| SCADA画面 | 需更新位号显示 |
| PLC程序 | 需更新变量命名 |
| 点表 | 需重新导出 |
