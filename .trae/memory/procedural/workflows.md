# 程序记忆

存储标准操作流程和最佳实践。

## 标准工作流程

### 任务规划流程
1. 理解目标 → 确认成功标准
2. 任务拆解 → 使用TodoWrite
3. 执行计划 → 按优先级排序
4. 执行验证 → 每步验证
5. 总结复盘 → 记录经验

### 规则改进流程
1. 问题识别 → 发现规则缺陷
2. 方案设计 → 制定改进方案
3. 规划文档 → 生成规划文档
4. 用户确认 → 确认后执行
5. 实施验证 → 更新规则文件

## 工具使用模式

| 场景 | 推荐工具 |
|------|----------|
| 读取配置 | Read |
| 搜索代码 | Grep |
| 修改文件 | SearchReplace |
| 跟踪任务 | TodoWrite |
| 执行命令 | RunCommand |

## 常见问题处理

- 路径问题：使用绝对路径
- 编码问题：UTF-8
- 权限问题：检查文件属性
