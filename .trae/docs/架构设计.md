## 一、系统概述

### 1.1 系统定位

本系统是一个**工业数字孪生演示平台**，用于展示工厂生产过程的实时监控、数据可视化、和仿真推演能力。

### 1.2 核心能力

| 能力 | 当前状态 | 目标状态 |
|------|---------|---------|
| 3D可视化 | 基础几何体 | PBR材质+精细模型 |
| 数据采集 | Mock静态数据 | Mock动态模拟 |
| 设备交互 | 仅查看 | 点击控制 |
| 报警管理 | 颜色变化 | 完整报警系统 |
| 物料追踪 | 无 | 端到端追踪 |
| 仿真推演 | 无 | 故障注入+培训 |

## 三、数据架构

### 3.1 数据流

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ Mock数据    │ ──► │  Simulator  │ ──► │   Store     │
│ 生成器      │     │  (100ms)    │     │  (Pinia)   │
└─────────────┘     └─────────────┘     └─────────────┘
                                               │
                    ┌────────────────────────────┤
                    ▼                            ▼
            ┌─────────────┐              ┌─────────────┐
            │  3D场景     │              │  Vue组件    │
            │  (Three.js) │              │  (模板绑定) │
            └─────────────┘              └─────────────┘
```

### 3.2 核心数据模型

#### 3.2.1 设备数据

```typescript
interface DeviceData {
  id: string              // 设备ID (如 'R-101')
  name: string            // 设备名称 (如 '反应釜')
  type: 'reactor' | 'tank' | 'valve' | 'pump' | 'conveyor'
  status: 'running' | 'stopped' | 'maintenance' | 'alarm'
  data: {
    temperature?: number  // 温度 (°C)
    pressure?: number     // 压力 (MPa)
    level?: number        // 液位 (%)
    flowRate?: number     // 流量 (m³/h)
    speed?: number         // 转速 (RPM)
  }
  position: { x: number, y: number, z: number }  // 3D位置
}
```

#### 3.2.2 报警数据

```typescript
interface Alarm {
  id: string
  deviceId: string
  level: number        // 1-提示, 2-注意, 3-警告, 4-严重, 5-紧急
  code: string           // 报警代码
  message: string        // 报警信息
  value?: number         // 触发值
  threshold?: number     // 阈值
  timestamp: Date
  acknowledged: boolean
  acknowledgedBy?: string
  acknowledgedAt?: Date
}
```

#### 3.2.3 物料批次

```typescript
interface MaterialLot {
  id: string              // 批次号
  materialId: string      // 物料编号
  materialName: string    // 物料名称
  quantity: number        // 数量
  unit: string            // 单位
  status: LotStatus        // 状态枚举
  currentLocation: string  // 当前位置
  batchDate: Date         // 生产日期
  history: LocationHistory[]  // 位置历史
}

enum LotStatus {
  IN_TRANSIT = 'in_transit'
  STORED = 'stored'
  IN_PROCESS = 'in_process'
  QC_PENDING = 'qc_pending'
  COMPLETED = 'completed'
}
```

## 四、模块设计

### 4.1 PlantSimulator (数据模拟引擎)

```javascript
class PlantSimulator {
  // 状态
  state: PlantState

  // 方法
  init()                    // 初始化
  start()                   // 启动模拟
  stop()                    // 停止模拟
  reset()                   // 重置状态

  // 模拟更新
  updateTemperature()        // 温度随机游走
  updatePressure()           // 压力随机游走
  updateLevel()              // 液位积分逻辑
  updateFlowRate()           // 流量正弦波动

  // 报警
  checkAlarms()             // 检查报警条件
  triggerAlarm(alarm)        // 触发报警
  acknowledgeAlarm(id)       // 确认报警

  // 监听
  addListener(callback)      // 添加状态监听
  removeListener(callback)   // 移除监听
}
```

### 4.2 Pinia Stores

| Store | 职责 | 核心状态 |
|-------|------|---------|
| plantStore | 工厂数据 | tankLevel, temperature, pressure, flowRate |
| plcStore | PLC数据 | motorSpeed, productCount, alarms |
| alarmStore | 报警管理 | activeAlarms, alarmHistory |
| perfStore | 性能监控 | fps, memory, dataThroughput |

### 4.3 Three.js Plant3D

```javascript
class Plant3D {
  // 场景
  scene: THREE.Scene
  camera: THREE.PerspectiveCamera
  renderer: THREE.WebGLRenderer

  // 控制
  controls: OrbitControls

  // 方法
  init()                     // 初始化场景
  createLights()             // 创建灯光
  createGround()             // 创建地面
  createReactor()            // 创建反应釜
  createTanks()              // 创建储罐
  createPipes()              // 创建管道
  createWorkers()            // 创建工人动画
  updateDeviceStatus(id, status)  // 更新设备状态
  focusOnDevice(id)          // 聚焦设备
  animate()                  // 动画循环
  dispose()                  // 清理资源
}
```

## 八、性能考量

### 8.1 渲染优化

| 优化项 | 策略 |
|--------|------|
| 3D渲染 | 使用 requestAnimationFrame，控制帧率 |
| 数据更新 | 100ms批量更新，避免频繁setState |
| 图表更新 | ECharts setOption 合并模式 |
| 组件渲染 | v-memo 缓存不变组件 |

### 8.2 内存管理

| 问题 | 解决方案 |
|------|---------|
| Three.js资源释放 | dispose() 清理几何体、材质 |
| 事件监听泄漏 | onUnmounted 移除监听 |
| 大数组 | 限制历史数据长度（100条） |
