#### 功能说明

| 功能 | 说明 | 交互 |
|------|------|------|
| 工艺流程总图 | 显示整条生产线设备状态，设备运行中绿色，停止灰色，故障红色 | 点击设备跳转工段画面 |
| 关键参数汇总 | 显示各工段关键工艺参数 | - |
| 今日生产状态 | 显示产量、合格率、运行时长、批次号、当前产品 | - |
| 报警信息 | 显示当前未确认的3条最高优先级报警 | 点击跳转报警画面 |
| 导航按钮 | 快速跳转到各工段画面 | 点击跳转 |

#### 数据刷新

| 数据类型 | 刷新周期 | 说明 |
|----------|----------|------|
| 设备状态 | 0.5s | 运行/停止/故障 |
| 工艺参数 | 1s | 温度、压力、液位等 |
| 产量数据 | 1s | 实时更新 |
| 报警信息 | 实时 | 有报警立即更新 |
