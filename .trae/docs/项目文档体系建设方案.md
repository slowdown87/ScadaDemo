## 3. 各类文档详细方案

### 3.1 配方管理（✅ 可自动生成）

#### 3.1.1 内容范围

配方管理涵盖以下三类配方：

| 配方类型 | 内容 | 应用场景 |
|----------|------|----------|
| **产品配方** | 茶叶原料配比、糖度、pH等 | 调配系统 |
| **工艺配方** | 萃取温度/时间、CIP程序参数 | 工艺控制 |
| **清洗配方** | CIP清洗剂浓度、温度、时间 | CIP系统 |

#### 3.1.2 数据来源

从现有文档提取：
- 产品方案边界说明.md — 产品类型、配方范围
- 工艺流程设计.md — 工艺参数
- CIP清洗程序规格书.md — 清洗程序

#### 3.1.3 输出格式

```yaml
# 产品配方示例
product_recipes:
  green_tea:
    name: "绿茶"
    blending:
      tea_concentration: "0.5-1.0%"
      brix: "8-10°Brix"
      ph: "5.5-6.0"
    extraction:
      temperature: "85-95℃"
      time: "10-15min"
      ratio: "1:50"
```

#### 3.1.4 自动生成能力

| 功能 | 可行性 | 说明 |
|------|--------|------|
| 配方参数YAML文件 | ✅ | 模板化管理 |
| 配方导出Excel | ✅ | 可自动生成 |
| 配方版本管理 | ✅ | 模板支持版本字段 |
| 配方对比表 | ✅ | 可自动生成 |

### 3.2 电控柜设计图（⚠️ 部分可行）

#### 3.2.1 内容范围

| 图纸类型 | 内容 | 自动生成可行性 |
|----------|------|---------------|
| 电气原理图 | 主回路、控制回路 | ❌ 无法生成CAD |
| 端子接线图 | 端子排定义、线号 | ✅ 可生成清单 |
| 柜内布置图 | 元器件排列 | ❌ 无法生成CAD |
| 电缆清单 | 电缆规格、起点终点 | ✅ 可生成清单 |
| I/O清单 | 模块配置、地址分配 | ✅ 已从点表生成 |

#### 3.2.2 可自动生成的内容

**元器件清单：**
```yaml
cabinet_config:
  main_panel:
    plc:
      - model: "S7-1500 R/H"
        cpu: "1517-3 PN/DP"
       DI: 1024点
        DO: 512点
        AI: 128点
        AO: 64点
    power_supply:
      - model: "SITOP PSU"
        rating: "24VDC, 10A"
        quantity: 2
    relay:
      - model: "FINDER 40.52"
        rating: "250VAC, 8A"
        coil: "24VDC"
        quantity: 50
```

**电缆清单（部分）：**
```yaml
cable_list:
  - cable_id: "C001"
    from: "PLC-1柜"
    to: "WT-101现场箱"
    type: "KVVP 4×1.5"
    length: 50
    purpose: "液位信号"
  - cable_id: "C002"
    from: "PLC-1柜"
    to: "WT-102现场箱"
    type: "KVVP 4×1.5"
    length: 55
    purpose: "流量信号"
```

#### 3.2.3 不能自动生成的内容

| 图纸类型 | 原因 |
|----------|------|
| 电气原理图 | 需要专业电气设计知识、CAD软件 |
| 柜内布置图 | 需要根据实际柜体尺寸规划 |
| 端子接线图 | 需要结合现场实际情况 |

#### 3.2.4 建议方案

**电控柜设计采用 EPLAN 或 AutoCAD Electrical，需要专业电气工程师完成。**

可提供的辅助文件：
- 元器件清单（Excel）
- I/O分配表（从监控点表派生）
- 电缆清单模板
- 端子排定义模板

### 3.3 现场仪表接线图（⚠️ 部分可行）

#### 3.3.1 内容范围

| 图纸类型 | 内容 | 自动生成可行性 |
|----------|------|---------------|
| 仪表位置图 | 仪表安装位置、标高 | ❌ 无法生成CAD |
| 仪表接线图 | 仪表端子定义、接线方式 | ✅ 可生成清单 |
| 仪表规格表 | 仪表型号、量程、精度 | ✅ 可从设备清单生成 |
| 桥架布置图 | 电缆桥架走向 | ❌ 无法生成CAD |

#### 3.3.2 可自动生成的内容

**仪表清单（从devices.yaml派生）：**
```yaml
instrument_list:
  - tag: "WT-101-LT"
    name: "原水箱液位"
    type: "超声波液位计"
    model: "E+H Prosonic FDU91"
    range: "0-10m"
    output: "4-20mA"
    power: "24VDC"
    installation: "顶部法兰安装"
    cable: "KVVP 2×1.5"
  - tag: "WT-101-TT"
    name: "原水箱温度"
    type: "PT100温度计"
    model: "E+H TMT182"
    range: "0-50℃"
    output: "4-20mA"
    power: "24VDC"
    insertion: "DN50法兰"
    cable: "KVVP 2×1.5"
```

**端子定义（部分）：**
```yaml
terminal_assignments:
  - terminal: "1-1"
    signal: "WT-101-LT"
    type: "AI"
    channel: "IW0"
    function: "液位信号 4-20mA"
  - terminal: "1-2"
    signal: "WT-101-TT"
    type: "AI"
    channel: "IW2"
    function: "温度信号 4-20mA"
```

#### 3.3.3 不能自动生成的内容

| 图纸类型 | 原因 |
|----------|------|
| 仪表位置图 | 需要现场勘查、CAD绘制 |
| 仪表接线图 | 需要专业仪表知识 |
| 桥架布置图 | 需要综合现场布局设计 |

#### 3.3.4 建议方案

**现场仪表图纸由仪表供应商或电气工程师根据仪表清单完成。**

可提供的辅助文件：
- 完整仪表清单（Excel）
- 仪表规格表
- 端子定义表
- 仪表电缆清单
