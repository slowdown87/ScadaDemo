## 1. 位号编码体系概述

### 1.1 编码原则

| 原则 | 说明 |
|------|------|
| 唯一性 | 每个位号唯一对应一个物理点或设备 |
| 可扩展性 | 序号编码留有余量，便于后续扩展 |
| 规律性 | 同类设备使用相同前缀和序号规则 |
| 可识别性 | 从位号可直观判断设备类型和工段 |

### 1.2 位号结构

```
[前缀][类型标识][序号]-[测量类型后缀]
```

| 组成部分 | 说明 | 示例 |
|----------|------|------|
| 前缀 | 工段标识(2-3字符) | WT, TH, EX, BL |
| 类型标识 | 设备类型(可选) | 无(储罐), V(阀门), P(泵), M(电机) |
| 序号 | 3位数字 | 101, 102 |
| 测量后缀 | 测量类型 | LT, TT, FT, ST |
