## 一、编码规范

### 1.1 JavaScript/TypeScript

#### 变量命名

| 类型 | 规则 | 示例 |
|------|------|------|
| 变量 | camelCase | `tankLevel`, `reactorTemp` |
| 常量 | UPPER_SNAKE_CASE | `MAX_TEMPERATURE`, `DEFAULT_TIMEOUT` |
| 函数 | camelCase | `updateTemperature()`, `checkAlarm()` |
| 类名 | PascalCase | `PlantSimulator`, `AlarmManager` |
| 布尔值 | is/has/can前缀 | `isRunning`, `hasAlarm`, `canControl` |

#### 文件命名

| 类型 | 规则 | 示例 |
|------|------|------|
| Vue组件 | PascalCase | `PlantView.vue`, `AlarmPanel.vue` |
| JS/TS文件 | camelCase | `plantStore.js`, `dataSimulator.ts` |
| 工具函数 | camelCase | `formatDate.js`, `calculateRate.ts` |

#### 组件内变量顺序

```javascript
export default {
  name: 'ComponentName',

  // 1. Props
  props: {
    title: String,
    data: {
      type: Array,
      default: () => []
    }
  },

  // 2. Components
  components: {
    ChildComponent
  },

  // 3. Composables (hooks)
  setup() {
    // 4. Reactive state
    const count = ref(0)
    const state = reactive({})

    // 5. Computed
    const doubled = computed(() => count.value * 2)

    // 6. Watchers
    watch(count, (newVal) => {
      console.log(newVal)
    })

    // 7. Lifecycle
    onMounted(() => {})

    // 8. Methods
    function increment() {
      count.value++
    }

    // 9. Return
    return {
      count,
      doubled,
      increment
    }
  }
}
```

## 三、Vue组件规范

### 3.1 组件结构

```vue
<template>
  <div class="component-name">
    <!-- 模板内容 -->
  </div>
</template>

<script setup>
// 1. 导入
import { ref, computed, onMounted } from 'vue'
import { useStore } from 'pinia'

// 2. Props定义
const props = defineProps({
  title: {
    type: String,
    required: true
  },
  data: {
    type: Array,
    default: () => []
  }
})

// 3. Emits定义
const emit = defineEmits(['update', 'delete'])

// 4. Composables
const { formatDate } = useDateUtils()

// 5. Store
const plantStore = usePlantStore()

// 6. Reactive
const localData = ref([])

// 7. Computed
const isEmpty = computed(() => props.data.length === 0)

// 8. Watchers
watch(() => props.data, (newVal) => {
  localData.value = newVal
})

// 9. Lifecycle
onMounted(() => {
  loadData()
})

// 10. Methods
function loadData() {
  // ...
}

function handleClick() {
  emit('update', localData.value)
}

// 11. Expose
defineExpose({ loadData })
</script>

<style scoped>
/* 样式 */
</style>
```

### 3.2 组件命名

| 类型 | 规则 | 示例 |
|------|------|------|
| 页面组件 | `XxxView.vue` | `HomeView.vue`, `TwinView.vue` |
| 布局组件 | `TheXxx.vue` | `TheHeader.vue`, `TheSidebar.vue` |
| 通用组件 | `XxxPanel.vue` | `AlarmPanel.vue`, `DataPanel.vue` |
| 基础组件 | `BaseXxx.vue` | `BaseButton.vue`, `BaseInput.vue` |

### 3.3 Props定义

```javascript
// 详细props定义
defineProps({
  // 基础类型
  title: String,

  // 多类型
  value: [String, Number],

  // 必填
  id: {
    type: String,
    required: true
  },

  // 默认值
  status: {
    type: String,
    default: 'idle'
  },

  // 数组/对象默认值
  items: {
    type: Array,
    default: () => []
  },
  config: {
    type: Object,
    default: () => ({})
  },

  // 自定义验证
  level: {
    type: Number,
    validator: (v) => v >= 0 && v <= 5
  }
})
```

## 四、Git规范

### 4.1 分支命名

| 类型 | 规则 | 示例 |
|------|------|------|
| 主分支 | `main` | `main` |
| 开发分支 | `develop` | `develop` |
| 功能分支 | `feature/xxx` | `feature/alarm-store` |
| 修复分支 | `fix/xxx` | `fix/nav-menu` |
| 文档分支 | `docs/xxx` | `docs/api` |

### 4.2 Commit规范

格式：`type(scope): subject`

```
type: feat | fix | docs | style | refactor | test | chore
```

示例：
```
feat(store): 添加alarmStore报警状态管理
fix(view): 修复SideNav导航不跳转问题
docs(readme): 更新部署说明
style(component): 统一AlarmPanel样式
refactor(three): 重构Plant3D灯光逻辑
test(store): 添加plantStore单元测试
chore(deps): 升级echarts到5.5.0
```

### 4.3 Commit频率

| 场景 | 建议 |
|------|------|
| 完成一个功能 | 立即commit |
| 完成一个bug修复 | 立即commit |
| 完成一天的编码 | 当天结束前commit |
| 离开电脑前 | 先commit再离开 |

## 六、代码质量

### 6.1 禁止事项

| 禁止 | 替代方案 |
|------|---------|
| `any` 类型 | 使用具体类型或 `unknown` |
| `console.log` 生产代码 | 使用日志模块或移除 |
| 硬编码魔数 | 定义常量 |
| 直接修改props | 使用emit或store |
| 内联样式 | 使用class或scoped style |
| `$refs` 操作DOM | 使用ref响应式变量 |

### 6.2 推荐实践

```javascript
// ✅ Good: 使用常量
const MAX_TEMPERATURE = 100
if (temp > MAX_TEMPERATURE) { }

// ✅ Good: 解构赋值
const { name, value } = props.data

// ✅ Good: 箭头函数
const handleClick = () => {
  emit('click')
}

// ✅ Good: 异步async/await
async function loadData() {
  const data = await fetchData()
  return data
}

// ✅ Good: 防御性编程
function divide(a, b) {
  if (b === 0) return 0
  return a / b
}
```

## 七、注释规范

### 7.1 JSDoc注释

```javascript
/**
 * 格式化温度值
 * @param {number} temp - 温度值（摄氏度）
 * @param {boolean} showUnit - 是否显示单位
 * @returns {string} 格式化后的温度字符串
 */
function formatTemperature(temp, showUnit = true) {
  const formatted = temp.toFixed(1)
  return showUnit ? `${formatted}°C` : formatted
}
```
