## 二、ECharts优化

### 2.1 高频更新策略

#### 问题
100ms间隔更新会导致频繁重绘，影响性能。

#### 解决方案：manual-update模式

```vue
<template>
  <v-chart
    ref="chartRef"
    :option="chartOption"
    :manual-update="true"
    autoresize
  />
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import VChart from 'vue-echarts'

const chartRef = ref(null)
const chartOption = ref({
  xAxis: { type: 'time' },
  yAxis: { type: 'value' },
  series: [{ type: 'line', data: [] }]
})

let updateTimer = null
const dataBuffer = []

onMounted(() => {
  updateTimer = setInterval(() => {
    if (dataBuffer.length > 0) {
      chartOption.value = {
        ...chartOption.value,
        series: [{
          ...chartOption.value.series[0],
          data: [...dataBuffer]
        }]
      }
      chartRef.value?.setOption(chartOption.value, { notMerge: false })
    }
  }, 100)
})

onUnmounted(() => {
  if (updateTimer) clearInterval(updateTimer)
})
</script>
```

## 三、Three.js优化

### 3.1 渲染循环优化

#### 问题
requestAnimationFrame每帧都渲染，即使没有变化。

#### 解决方案：脏检查

```javascript
class Plant3D {
  constructor() {
    this.needsUpdate = false
    this.frustumDirty = false
    this.animationId = null
  }

  markDirty(type = 'all') {
    this.needsUpdate = true
    if (type === 'frustum') this.frustumDirty = true
  }

  animate() {
    this.animationId = requestAnimationFrame(() => this.animate())

    if (this.needsUpdate) {
      if (this.controls) this.controls.update()
      if (this.frustumDirty) this.updateFrustum()
      this.renderer.render(this.scene, this.camera)
      this.needsUpdate = false
      this.frustumDirty = false
    }
  }

  dispose() {
    if (this.animationId) {
      cancelAnimationFrame(this.animationId)
    }
  }
}
```

### 3.2 灯光优化

| 灯光类型 | 数量限制 | 性能影响 |
|----------|----------|----------|
| AmbientLight | 1个 | 低 |
| HemisphereLight | 1个 | 低 |
| DirectionalLight | ≤3个 | 中 |
| PointLight | ≤8个 | 高 |
| SpotLight | ≤4个 | 很高 |

```javascript
setupLights() {
  // 环境光 - 必须
  const ambient = new THREE.AmbientLight(0xffffff, 0.5)
  this.scene.add(ambient)

  // 半球光 - 提供自然光照
  const hemi = new THREE.HemisphereLight(0xadd8e6, 0x808080, 0.6)
  this.scene.add(hemi)

  // 主方向光 - 投射阴影
  const main = new THREE.DirectionalLight(0xffffff, 0.8)
  main.position.set(50, 100, 50)
  main.castShadow = true
  main.shadow.mapSize.width = 1024  // 降低阴影分辨率
  main.shadow.mapSize.height = 1024
  main.shadow.camera.near = 0.5
  main.shadow.camera.far = 500
  this.scene.add(main)

  // 避免添加更多点光源
}
```

## 五、内存管理

### 5.1 Three.js资源释放

```javascript
class Plant3D {
  dispose() {
    // 1. 停止动画循环
    if (this.animationId) {
      cancelAnimationFrame(this.animationId)
    }

    // 2. 清理控制器
    if (this.controls) {
      this.controls.dispose()
    }

    // 3. 清理几何体
    this.scene.traverse((object) => {
      if (object.geometry) {
        object.geometry.dispose()
      }

      // 4. 清理材质
      if (object.material) {
        if (Array.isArray(object.material)) {
          object.material.forEach(m => m.dispose())
        } else {
          object.material.dispose()
        }
      }
    })

    // 5. 清理纹理
    this.textures?.forEach(texture => texture.dispose())

    // 6. 清理渲染器
    if (this.renderer) {
      this.renderer.dispose()
      this.renderer.forceContextLoss()
    }

    // 7. 清理场景
    this.scene.clear()
  }
}
```

## 八、性能监控

### 8.1 FPS监控

```javascript
class PerformanceMonitor {
  constructor() {
    this.frames = 0
    this.lastTime = performance.now()
    this.fps = 0
  }

  update() {
    this.frames++
    const now = performance.now()
    const delta = now - this.lastTime

    if (delta >= 1000) {
      this.fps = Math.round((this.frames * 1000) / delta)
      this.frames = 0
      this.lastTime = now
      console.log(`FPS: ${this.fps}`)
    }
  }
}
```

### 8.2 内存监控

function logMemoryUsage() {
  if (performance.memory) {
    const used = (performance.memory.usedJSHeapSize / 1048576).toFixed(2)
    const total = (performance.memory.totalJSHeapSize / 1048576).toFixed(2)
    console.log(`Memory: ${used}MB / ${total}MB`)
  }
}
