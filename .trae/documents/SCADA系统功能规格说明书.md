# SCADA系统功能规格说明书

> 文档版本: v1.0
> 创建日期: 2026-04-28
> 项目名称: 茶饮料生产线SCADA监控系统
> 产能: 54000瓶/小时（500ml PET瓶）

---

## 1. 项目概述

### 1.1 项目背景

本项目为茶饮料（纯茶）生产线配套的SCADA（Supervisory Control and Data Acquisition）监控系统，用于实现对整条生产线的集中监控、数据采集、报警管理和生产报表功能。

### 1.2 项目范围

| 范围 | 说明 |
|------|------|
| 生产线 | 54000B/H PET瓶热灌装茶饮料生产线 |
| 监控对象 | 水处理、萃取、过滤、调配、均质、UHT杀菌、灌装、冷却、包装系统 |
| 控制方式 | 监控为主，关键设备联动控制 |
| 网络架构 | 集中监控，分布式I/O |

### 1.3 设计依据

| 标准/规范 | 说明 |
|----------|------|
| GB/T 21102-2007 | 过程工业中电气/仪表自动化系统的工程规范 |
| GB/T 25928-2010 | 过程工业自动化系统工程文件规范 |
| IEC 61131-3 | 可编程控制器编程语言 |
| ISA-5.1 | 仪表符号和标注标准 |
| GB/T 34050 | 智能工厂通用技术要求 |

### 2.2 网络架构

| 网络层级 | 协议 | 带宽 | 设备 |
|----------|------|------|------|
| 监控网络 | Ethernet TCP/IP | 1Gbps | 服务器、操作站 |
| 控制网络 | Ethernet/IP 或 PROFINET | 100Mbps | PLC、I/O |
| 现场总线 | Modbus RTU/TCP | 115.2kbps | 智能仪表 |

### 2.3 硬件配置

| 设备 | 规格 | 数量 | 备注 |
|------|------|------|------|
| SCADA服务器 | 32GB RAM, 2TB SSD, 双网口 | 1 | 安装历史数据库 |
| 操作站 | 16GB RAM, 512GB SSD | 2 | 主/备操作 |
| 工程师站 | 16GB RAM, 512GB SSD | 1 | 组态编程 |
| 核心交换机 | 24口工业交换机 | 1 | 冗余配置 |
| PLC CPU | S7-1500 冗余 | 3 | 各控制区 |
| I/O模块 | ET200SP | 若干 | 分布式 |

## 3. 功能规格

### 3.1 监控功能

#### 3.1.1 工艺流程画面

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 总貌画面 | 整条生产线工艺流程总览，关键参数集中显示 | P0 |
| 水处理系统画面 | 水处理8个工段的详细监控 | P1 |
| 萃取系统画面 | 三级逆流萃取的控制与监控 | P1 |
| 过滤调配画面 | 过滤、调配、糖浆制备监控 | P1 |
| UHT杀菌画面 | 温度、压力、流量实时监控 | P0 |
| 灌装系统画面 | 灌装、旋盖、冷却监控 | P0 |
| CIP清洗画面 | 清洗程序控制与状态监控 | P1 |
| 趋势画面 | 历史趋势曲线，支持30天数据 | P1 |

#### 3.1.2 数据显示

| 功能 | 描述 | 刷新周期 |
|------|------|----------|
| 实时数据 | 温度、压力、液位、流量等 | 1秒 |
| 设备状态 | 运行/停止/故障状态 | 0.5秒 |
| 报警状态 | 当前报警列表 | 实时 |
| 产量数据 | 班次、日、月产量 | 实时 |

#### 3.1.3 动画效果

| 功能 | 描述 | 触发条件 |
|------|------|----------|
| 管道流动 | 液体流动动画 | 设备运行时 |
| 液位变化 | 储罐液位动态显示 | 液位变化时 |
| 阀门状态 | 开关状态颜色变化 | 状态变化时 |
| 泵/电机运行 | 旋转动画 | 运行时 |
| 报警闪烁 | 红色/黄色闪烁 | 报警触发时 |

### 3.2 报警功能

#### 3.2.1 报警等级

| 等级 | 名称 | 颜色 | 声音 | 处理方式 |
|------|------|------|------|----------|
| HH | 高高报警 | 红色闪烁 | 连续 | 立即处理，联锁动作 |
| H | 高报警 | 黄色 | 断续 | 30分钟内处理 |
| L | 低报警 | 黄色 | 断续 | 30分钟内处理 |
| LL | 低低报警 | 红色闪烁 | 连续 | 立即处理，联锁动作 |
| D | 偏差报警 | 蓝色 | 无 | 提示 |
| F | 故障报警 | 红色 | 断续 | 故障排除 |

#### 3.2.2 报警功能

| 功能 | 描述 |
|------|------|
| 报警列表 | 按时间、等级、设备过滤显示 |
| 报警确认 | 操作员确认报警 |
| 报警历史 | 存储90天报警记录 |
| 报警统计 | 报警次数、时长统计 |
| 报警打印 | 重要报警自动打印 |
| 短信通知 | 重大报警发送短信（选配） |

### 3.3 控制功能

#### 3.3.1 自动控制

| 功能 | 描述 | 方式 |
|------|------|------|
| 液位控制 | 储罐液位自动控制 | PID |
| 温度控制 | 萃取、调配温度控制 | PID |
| 流量控制 | 管道流量控制 | PID |
| pH控制 | 调配pH值控制 | PID |
| Brix控制 | 糖度控制 | PID |
| 压力控制 | 均质、管道压力控制 | PID |

#### 3.3.2 顺序控制

| 功能 | 描述 | 触发 |
|------|------|------|
| 批次控制 | 调配批次逻辑 | 配方选择 |
| 萃取控制 | 三级逆流萃取逻辑 | 自动/手动 |
| CIP程序 | 清洗程序自动执行 | 配方选择 |
| 灌装节奏 | 灌装速度同步控制 | 联动 |

#### 3.3.3 设备控制

| 功能 | 描述 | 权限 |
|------|------|------|
| 泵启停 | 单台泵的启停控制 | 操作员 |
| 阀开关 | 阀门开闭控制 | 操作员 |
| 电机控制 | 电机启停/转速调节 | 工程师 |
| 配方选择 | 生产配方选择 | 工艺员 |
| 参数调整 | 控制参数修改 | 工程师 |

### 3.4 数据管理

#### 3.4.1 数据采集

| 功能 | 描述 |
|------|------|
| 实时采集 | 模拟量、开关量、计数器 |
| 数据存储 | 实时数据、历史数据 |
| 数据压缩 | 旋转门压缩算法 |
| 数据校验 | CRC校验 |

#### 3.4.2 历史数据

| 功能 | 描述 |
|------|------|
| 存储周期 | 模拟量30天，开关量90天 |
| 存储间隔 | 模拟量1分钟（可调） |
| 数据查询 | 时间范围、设备、参数 |
| 数据导出 | Excel、CSV格式 |

#### 3.4.3 报表功能

| 报表类型 | 内容 | 生成方式 |
|----------|------|----------|
| 日报表 | 班次产量、工艺参数、设备运行时间 | 自动 |
| 周报表 | 周产量、质量统计、OEE | 自动 |
| 月报表 | 月度产量、质量、能耗分析 | 自动 |
| 年报表 | 年度生产统计、设备评估 | 自动 |
| 自定义报表 | 用户定义报表格式 | 手动/自动 |

## 5. 通讯规格

### 5.1 PLC通讯

| 项目 | 规格 |
|------|------|
| 通讯协议 | Ethernet/IP 或 S7-TCP/IP |
| 扫描周期 | 100ms |
| 通讯冗余 | 双网口冗余 |

| 糖度计 | Modbus RTU | 4 |
| 电导率仪 | Modbus RTU | 6 |
| 温度变送器 | PT100 | 28 |

### 5.3 第三方系统接口

| 系统 | 接口类型 | 数据 |
|------|----------|------|
| MES系统 | OPC UA | 产量、工艺参数 |

| 瓶盖供应 | 串口 | 计数信号 |
| 贴标机 | TCP/IP | 速度信号 |
