### 3.2 水处理系统流程

```
原水 → 原水箱 → 多介质过滤器 →活性炭过滤器 → RO反渗透 → 离子交换柱 → UV杀菌 → 纯水储罐
              (10m³)       (自动反洗)    (5m³/h)      (5m³/h)    (10m³/h)
```

### 3.3 萃取系统流程

```
茶叶 → 储罐 → 粉碎机(60目) → 萃取罐组(三级逆流) → 茶渣分离器 → 茶汁冷却器
                                              ↓
                                         茶渣排放 → 废渣处理
```

### 3.4 过滤净化系统流程

```
茶汁(80℃) → 碟式离心机(预过滤) → 超滤膜(0.1μm) → 茶浓缩汁
```

### 3.5 调配系统流程

```
茶浓缩汁 ─────────┐
                 ├──→ 调配罐 ──→ 均质机 ──→ UHT杀菌
糖浆 ◄──糖浆罐 ─┤
                 ├──→ 搅拌混合(15min)
酸液 ◄──酸液罐 ─┤
                 │
香精 ◄──香料罐 ─┘
```

### 3.6 UHT杀菌系统流程

```
调配液(70℃) → 预热段(80℃) → 高温杀菌段(135℃/15s) → 保温段 → 冷却段(25℃) → 无菌储罐
```

### 3.7 灌装系统流程

```
PET瓶坯 → 吹瓶机 → 空瓶输送 → 冲洗站 → 灌装机(72头) → 旋盖机(12头) → 冷却隧道
                                       (85-88℃)              ↓
                                                         灯检机 → 贴标机 → 喷码机 → 装箱机 → 码垛机
```

## 6. 工艺流程说明

### 6.1 水处理系统
水处理是茶饮料生产的基础，采用多级过滤确保水质纯净。原水经过砂滤去除大颗粒杂质，活性炭吸附有机物和余氯，RO反渗透去除溶解性固体，最后通过UV杀菌确保微生物达标。

### 6.2 茶叶萃取系统
茶叶萃取是核心工艺，采用三级逆流萃取技术，最大化提取茶多酚和风味物质。逆流萃取可使茶水比降低30%以上，节约用水和能源。

### 6.3 过滤净化系统
过滤系统采用先粗后精的原则，先去除茶渣和悬浮物，再通过超滤膜去除导致浑浊的大分子物质，确保茶饮料的澄清度和透明度。

### 6.4 调配系统
调配是保证产品一致性的关键工序。通过精确计量和搅拌，使每批产品的糖度、pH值、风味物质含量保持一致。

### 6.5 均质系统
均质处理可以细化茶饮料中的颗粒，防止沉淀分层，提高产品的稳定性和口感。

### 6.6 UHT杀菌系统
超高温瞬时杀菌可以在最短时间内杀灭微生物，同时最大程度保留茶叶的风味和营养成分。135℃/15s是茶饮料的标准杀菌参数。

### 6.7 灌装系统
采用72头回转式灌装机，产能达54000瓶/小时。热灌装方式可以在灌装同时完成产品消毒，延长保质期。

### 6.8 冷却包装系统
灌装后产品通过冷却隧道降温，然后经过贴标、喷码、装箱、码垛完成最终包装。

## 7. 卫生控制

### 7.1 CIP清洗程序

| 步骤 | 介质 | 温度 | 时间 | 浓度 |
|------|------|------|------|------|
| 1.预冲洗 | 热水 | 85℃ | 15min | - |
| 2.碱洗 | NaOH | 85℃ | 30min | 2% |
| 3.中间冲洗 | 纯水 | 常温 | 10min | - |
| 4.酸洗 | HNO₃ | 60℃ | 20min | 1% |
| 5.最终冲洗 | 纯水 | 常温 | 10min | pH6.5-7.0 |

### 7.2 生产环境要求

| 区域 | 洁净度 | 温度 | 湿度 | 压差 |
|------|--------|------|------|------|
| 灌装间 | 10万级 | 20-25℃ | ≤60% | ≥10Pa |
| 调配间 | 30万级 | 20-25℃ | ≤65% | 正压 |
| 一般区 | 普通 | 20-28℃ | ≤75% | - |
