# 茶饮料生产线SCADA系统联锁逻辑说明书

> 文档版本: v1.0
> 创建日期: 2026-04-28
> 项目名称: 茶饮料生产线SCADA监控系统
> 产能: 54000瓶/小时（500ml PET瓶）

---

## 1. 概述

### 1.1 文档目的

本文档定义茶饮料生产线SCADA系统中所有安全联锁和操作联锁的逻辑关系，作为PLC程序开发和SCADA组态的依据。

### 1.2 联锁分类

| 类别 | 说明 | 响应要求 |
|------|------|----------|
| 安全联锁(ESD) | 涉及人员安全、设备安全的紧急联锁 | 立即执行，无需确认 |
| 关键控制联锁 | 食品安全、产品质量关键点的联锁 | 立即执行，报警提示 |
| 操作联锁 | 防止误操作的程序联锁 | 禁止执行，提示原因 |
| 顺序联锁 | 批次生产、程序切换的逻辑联锁 | 按逻辑执行 |

### 1.3 联锁级别定义

| 级别 | 名称 | 颜色 | 声音 | 响应时间 | 处理方式 |
|------|------|------|------|----------|----------|
| L0 | 紧急停止 | 红色闪烁 | 连续 | ≤100ms | 立即停止所有相关设备 |
| L1 | 安全联锁 | 红色 | 断续 | ≤500ms | 停止故障设备及上游设备 |
| L2 | 关键报警 | 红色/黄色 | 断续 | ≤1s | 停止生产线，报警提示 |
| L3 | 工艺报警 | 黄色 | 无 | ≤1s | 报警提示，允许手动处理 |
| L4 | 提示信息 | 蓝色 | 无 | - | 显示记录，无需动作 |

## 4. 萃取系统联锁

### 4.1 萃取控制逻辑

三级逆流萃取采用批次控制，每级萃取罐独立温控。

### 4.2 萃取联锁因果表

| 序号 | 触发条件 | 报警级别 | 联锁动作 | 复位条件 |
|------|----------|----------|----------|----------|
| EX-01 | 萃取罐温度TT < 75℃ | LL (L2) | 1.开大蒸汽阀EV-10X<br>2.报警提示 | 温度>78℃后确认 |
| EX-02 | 萃取罐温度TT > 100℃ | HH (L1) | 1.关闭蒸汽阀EV-10X<br>2.开大进水阀<br>3.报警提示 | 温度<95℃后确认 |
| EX-03 | 萃取罐液位LT > 98% | HH (L1) | 1.关闭进水阀EV-101/104<br>2.关闭进茶阀EV-103/106<br>3.报警提示 | 液位<95%后自动 |
| EX-04 | 萃取罐液位LT < 5% | LL (L2) | 1.关闭出水阀EV-106<br>2.报警提示 | 液位>10%后确认 |
| EX-05 | 萃取时间到但温度未达标 | H (L3) | 1.延长萃取时间<br>2.报警提示 | 温度达标后继续 |
| EX-06 | 茶渣分离器堵塞 | PT差压高 | 1.停P-EX01<br>2.打开EV-107排放<br>3.报警提示 | 人工确认 |

### 4.3 萃取顺序控制

```
启动顺序:
1. 检查上一级茶汁储量 > 20%
2. 开萃取罐排水阀EV-106，液位<10%后关闭
3. 开进水阀EV-101，液位达到90%后关闭
4. 开进茶阀EV-103，进茶量达到设定值后关闭
5. 启动搅拌电机M-EX01，设定转速
6. 启动温度控制，TT达到设定值后开始计时
7. 萃取计时结束后，打开EV-106泵送茶汁
8. 循环直到本级萃取液送完

批次切换:
- 一级→二级: 二级茶汁已送完，一级新批次开始
- 二级→三级: 三级茶汁已送完，二级新批次开始
- 三级→排放: 三级茶汁送完后，排放茶渣
```

## 12. 联锁复位与确认

### 12.1 复位权限

| 级别 | 复位权限 | 确认要求 |
|------|----------|----------|
| L0 紧急停止 | 工程师/管理员 | 必须现场确认 |
| L1 安全联锁 | 工程师 | 必须现场确认 |
| L2 关键报警 | 工艺员/班长 | 操作站确认 |
| L3 工艺报警 | 操作员 | 操作站确认 |

### 12.2 复位流程

```
1. 消除触发原因
2. 检查设备状态
3. 确认安全条件
4. 在操作站复位报警
5. 记录复位时间和原因
6. 逐步恢复生产
```
