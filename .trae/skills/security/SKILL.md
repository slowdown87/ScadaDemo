# 安全审计技能

全面检查代码安全隐患，提供加固方案。

## 常见漏洞类型

### OWASP Top 10

| 漏洞 | 说明 | 检查重点 |
|-----|------|---------|
| **注入** | SQL/NoSQL/命令注入 | 用户输入处理 |
| **身份认证** | 弱密码/会话管理 | 认证机制 |
| **敏感泄露** | API Key/密码硬编码 | 配置和代码 |
| **XML外部实体** | XXE | 文件上传/解析 |
| **访问控制** | 越权访问 | 权限校验 |
| **安全配置** | 默认配置/错误配置 | 配置文件 |
| **XSS** | 跨站脚本 | 输出编码 |
| **CSRF** | 跨站请求伪造 | Token校验 |
| **反序列化** | 恶意反序列化 | 输入验证 |
| **日志监控** | 缺失审计 | 日志记录 |

## 安全检查流程

```
1. 资产识别
   - 敏感数据有哪些？
   - 关键接口是什么？
   - 信任边界在哪？

2. 威胁建模
   - 可能的攻击向量
   - 攻击者能力假设
   - 攻击收益分析

3. 漏洞扫描
   - 代码静态分析
   - 配置检查
   - 依赖检查

4. 风险评估
   - 严重程度
   - 利用难度
   - 影响范围
```

## 检查清单

### 输入验证
- [ ] 所有用户输入都验证？
- [ ] 白名单校验？
- [ ] 长度限制？
- [ ] 类型校验？

### 认证授权
- [ ] 强密码策略？
- [ ] 会话超时？
- [ ] 权限最小化？
- [ ] 敏感操作二次验证？

### 数据保护
- [ ] 敏感数据加密存储？
- [ ] 传输加密（TLS）？
- [ ] 日志脱敏？
- [ ] Key用环境变量？

### 错误处理
- [ ] 不暴露堆栈？
- [ ] 统一错误格式？
- [ ] 记录审计日志？
