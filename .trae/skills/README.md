# Skills 索引

按开发流程分类的技能清单。

## 使用顺序（开发流程）

```
需求 → 设计 → 实现 → 审查 → 部署
```

## 分类索引

### 分析类 - 需求/设计阶段

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **guided-ai** | 深度分析、多元方案 | "深入分析这个问题" |
| **multi-agent-sim** | 多角色协作模拟 | "用多角色模式分析" |

### 规划类 - 项目启动

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **planning** | 任务拆解、ReAct循环 | "激活规划模式" |

### 架构类 - 设计阶段

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **architecture** | 系统架构、技术选型 | "帮我设计架构" |

### 编码类 - 实现阶段

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **swe-agent-mode** | 修Bug、代码修复 | "修复这个Bug" |

### 质量类 - 审查阶段

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **code-review** | 代码审查、发现问题 | "审查这段代码" |
| **security** | 安全审计、漏洞扫描 | "安全审查" |
| **testing** | 测试策略、测试生成 | "设计测试策略" |

### 维护类 - 修复/优化

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **debug** | 调试定位、问题追踪 | "帮我调试" |
| **refactoring** | 代码重构、坏味道消除 | "需要重构" |

### 文档类 - 全流程

| 技能 | 说明 | 激活词 |
|-----|------|-------|
| **documentation** | 文档生成与维护 | "写文档" |

---

## 快速参考

| 场景 | 推荐技能 |
|-----|---------|
| 需求分析 | guided-ai + planning |
| 架构设计 | architecture + multi-agent-sim |
| 写代码 | swe-agent-mode |
| 修Bug | swe-agent-mode + debug |
| 代码审查 | code-review + security |
| 写测试 | testing |
| 重构 | refactoring + testing |
| 写文档 | documentation |

## 激活方式

当需要使用某个技能时，可以说：
- "激活 planning 技能"
- "用多角色模式分析"
- "帮我审查这段代码"

或者让 AI 根据场景自动选择合适的技能。
