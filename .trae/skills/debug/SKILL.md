# 调试技能

系统性调试，快速定位和修复Bug。

## 调试流程

```
1. 复现问题
   - 确认Bug表现
   - 收集错误信息
   - 最小化复现场景

2. 分析问题
   - 追踪调用链
   - 检查数据流
   - 定位根因

3. 制定修复
   - 设计解决方案
   - 评估影响范围
   - 准备备选方案

4. 验证修复
   - 复现Bug是否消除
   - 是否有副作用
   - 边界条件是否正常
```

## 调试策略

### 日志分析
```
- 查找错误关键字
- 追踪时间线
- 检查异常堆栈
```

### 代码追踪
```
- 入口点 → 中间变量 → 出口点
- 使用 grep/find 定位关键代码
```

### 二分定位
```
- 注释一半代码
- 运行测试
- 确定问题在半区的哪一侧
- 递归直到定位
```

## 常见Bug模式

| 类型 | 特征 | 定位方法 |
|-----|------|---------|
| **空指针** | NullReferenceException | 追踪对象初始化 |
| **数组越界** | IndexOutOfRange | 检查索引计算 |
| **并发问题** | 随机失败 | 检查锁/状态共享 |
| **异步问题** | 时序错误 | 检查回调/await |
| **配置错误** | 环境差异 | 检查环境变量 |

- [ ] 问题可复现吗？

- [ ] 修复会引入新问题吗？
- [ ] 测试覆盖了吗？
- [ ] 文档更新了吗？
